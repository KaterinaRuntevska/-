#-# Апликација за менаџирање на студентски оценки 🎓

Ова е едноставна JavaScript апликација која работи во терминал и овозможува:
- Внесување на студенти и нивни оценки.
- Автоматска проверка на валидност на оценките (1-10).
- Пресметување просек за секој студент.
- Истакнување на најдобриот студент во групата.

### Како се стартува:
1. Инсталирајте зависности: `npm install`
2. Стартувајте ја програмата: `node index.js`
